About This Application

My Spring Boot REST service built with modern web technologies and deployed to the cloud.

What This App Does

  • REST API: Provides greeting endpoints that return JSON data
  • Web Interface: Beautiful web pages built with Thymeleaf templates
  • Cross-Origin Support: Can be used by any frontend application
  • Cloud Deployed: Running on AWS with SSL encryption

Technical Features

  • Spring Boot 3.5.5: Modern Java framework
  • Thymeleaf: Server-side template engine
  • Gradle: Build automation tool
  • Nginx: Reverse proxy and SSL termination
  • Supervisor: Process management and auto-restart
  • Bootstrap 5: Responsive UI framework
